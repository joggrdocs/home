# Coding Standards PR Check

**Status:** ![Planned](https://img.shields.io/badge/Planned-%23003366)

## Summary

Detect stale coding standards docs and related coding standards rules on every pull request.

## Problem

Coding standards documents and their associated rules drift out of date silently, causing agents and developers to follow outdated practices without any warning at review time.

## Solution

Coding Standards PR Check runs automatically on every pull request to detect stale coding standards documents and mismatched rules, surfacing issues before code is merged.

## Impact

Teams maintain consistently up-to-date coding standards, ensuring that both human reviewers and AI agents operate from the same authoritative guidelines on every pull request.
