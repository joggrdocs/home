# Coding Agent Setup Remediation

## Problem

After Joggr identifies misconfigured or missing AI development settings, users must still research and apply each fix, which is time-consuming and manual.

## What we're releasing

Coding Agent Setup Remediation automatically generates fixes from Joggr’s findings and packages them into a pull request for developer review—removing the need for manual intervention while maintaining a human in the loop review.

## Expected outcome

Users can resolve AI development setup issues quickly, ensuring their repositories stay properly configured with minimal effort.
