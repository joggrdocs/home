# GG Workspaces: Remote

## Problem

Local execution of AI agents still ties agent workloads to a developer's machine, limiting concurrency and requiring the developer to remain online for the duration of long-running tasks.

## What we're releasing

GG Workspaces: Remote offloads agent execution to isolated remote sandboxes, enabling fully autonomous operation without any dependency on the developer's local machine.

## Expected outcome

Teams can run AI agents at scale in the background, freeing developers to focus on other work while agents autonomously complete tasks in a secure, isolated environment.
