# Coding Standards Generation

**Status:** ![Planned](https://img.shields.io/badge/Planned-%23003366)

## Summary

Generate a codebase-aware set of standards docs based on detected patterns and conventions, and include in the AI instructions and rules as needed.

## Problem

Most codebases have implicit conventions that are never formally documented, causing AI coding agents to produce inconsistent code that doesn't match the project's established patterns.

## Solution

Coding Standards Generation analyzes the existing codebase to detect patterns and conventions, then produces standards documentation that is automatically wired into AI agent instruction and rule files.

## Impact

AI coding agents produce code that is stylistically and structurally consistent with the existing codebase, reducing review friction and maintaining code quality at scale.
