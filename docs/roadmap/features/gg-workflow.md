# GG Workflow

**Status:** ![Planned](https://img.shields.io/badge/Planned-%23003366)

## Summary

A structured workflow for developing with AI (discuss → research → plan → execute → verify), built using Agent Skills and a custom CLI.

## Problem

Developers using AI coding assistants lack a repeatable, structured process, leading to inconsistent results and wasted effort when tackling complex tasks.

## Solution

GG Workflow introduces a step-by-step development workflow (discuss, research, plan, execute, verify) powered by Agent Skills and a custom CLI, ensuring each phase builds on the last for reliable outcomes.

## Impact

Teams get a predictable, high-quality AI development process that reduces rework and helps developers confidently go from problem statement to verified solution.
