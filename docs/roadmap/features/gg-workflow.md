# GG: Structure AI Dev Workflow

## Problem

Developers using AI coding assistants lack a repeatable, structured process, leading to inconsistent results and wasted effort when tackling complex tasks. As work grows more complex, they also run into context window limits, causing earlier discussions, plans, and decisions to fall out of context and forcing developers to manually re-explain them.

## What we're releasing

GG (a Joggr CLI tool) Workflow introduces a structured development workflow (discuss, research, plan, execute, verify) powered by Agent Skills and a custom CLI. Each phase captures discussions, research, and plans as markdown files that feed into the agent’s memory, preserving context and progress across the entire task.

## Expected outcome

Developers can tackle complex tasks with AI in a structured way while preserving context across long workflows, reducing repeated work and improving consistency.
