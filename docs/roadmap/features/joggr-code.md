# The Joggr Coding Agent

## Problem

Even with individual components like agent harnesses, skill registries, and documentation platforms in place, developers still need a unified coding agent that ties everything together into a seamless workflow.

## What we're releasing

Joggr Code is a full coding agent that integrates the agent harness, global and private skill registries, and Vibepress documentation platform into a single, cohesive development experience.

## Expected outcome

Developers get an end-to-end AI coding assistant backed by the entire Joggr ecosystem, enabling faster and more reliable software delivery out of the box.
