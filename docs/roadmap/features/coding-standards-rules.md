# Coding Standards Rules

## Problem

Coding standards documents drift out of sync with the rule and instruction files that AI agents actually consume, leading agents to ignore or contradict the team's documented conventions.

## What we're releasing

Coding Standards Rules automatically generates and keeps agent rule and instruction files in sync with the source coding standards documentation, ensuring agents always follow the latest conventions.

## Expected outcome

Teams maintain a single source of truth for coding standards while AI agents consistently follow those standards, reducing manual upkeep and preventing convention drift.
