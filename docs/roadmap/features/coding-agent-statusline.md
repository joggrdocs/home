# Coding Agent Statusline

**Status:** ![Approved](https://img.shields.io/badge/Approved-%23006B3F)

## Summary

Displays repository state and AI session context in a terminal status bar, including GG workflow state, optimized for Claude Code and other terminal-based coding agents.

## Problem

Developers using terminal-based coding agents lack at-a-glance visibility into the current repository state and active AI session context, forcing them to run manual commands to understand what the agent is working on and where things stand.

## Solution

Coding Agent Statusline adds a persistent terminal status bar that surfaces repository state, GG workflow status, and AI session context in real time, purpose-built for coding agent environments.

## Impact

Developers maintain continuous awareness of agent activity and repository state without interrupting their workflow, leading to faster iteration and fewer context-switching mistakes.
