# Context MCP: Internal

**Status:** ![Approved](https://img.shields.io/badge/Approved-%23006B3F)

## Summary

Aggregates repository structure, documentation, and project metadata into structured context for agents.

## Problem

Agents operating on a codebase lack a consolidated view of repository structure, internal documentation, and project metadata, leading to incomplete understanding and lower-quality outputs.

## Solution

Context MCP: Internal aggregates repository structure, documentation, and project metadata into a single structured context source that agents can query before performing tasks.

## Impact

Agents make better-informed decisions by starting every task with a complete picture of the project, resulting in more accurate code generation and fewer errors rooted in missing context.
