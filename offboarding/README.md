# Offboarding Joggr

Pick the Joggr product you're offboarding from:

- **[Joggr coding](./coding/)** — the CLI (`@joggr/cli`, `jog`/`joggr` binaries), Claude Code hook, and per-machine skills/agents.
- **[Joggr docs](./docs/)** — the GitHub App install plus the Joggr-managed HTML comments injected into your markdown files.

Each subdirectory ships its own scripts (`offboard.mjs`, `status.mjs`, `restore.mjs`), `package.json` with npm scripts, and a `README.md` walking through both a safe manual path and an automatic path.

License terms live at [`./LICENSE.md`](./LICENSE.md) and apply to everything in this directory.
